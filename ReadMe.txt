How to  run 
---------------
1) go to folder from comad prompt
2)enter commad ----> npm install
3) then ---> ionic serve --open

how to build
----------------
1) go to folder from comad prompt
2) enter commad ---> ionic cordova build android
3) will find the apk in the folder ---> *project folder*\platforms\android\app\build\outputs\apk\debug

Editing Css

eg: login page --> login folder --> login.scss
